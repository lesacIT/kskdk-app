IV. HƯỚNG DẪN CÀI ĐẶT

4.1. Yêu cầu hệ thống

  • Node.js phiên bản 14.x trở lên
  • npm phiên bản 6.x trở lên
  • Trình duyệt web hiện đại (Chrome, Firefox, Edge)

4.2. Các bước cài đặt

Bước 1: Tải mã nguồn

git clone <repository-url>
cd kskdk-app

Bước 2: Cài đặt dependencies cho backend

cd backend
npm install

Bước 3: Cấu hình biến môi trường

Tạo file .env trong thư mục backend với nội dung:

PORT=3000
NODE_ENV=development
DB_PATH=./kskdk.db
CORS_ORIGIN=*

Bước 4: Khởi chạy ứng dụng

npm run dev

Bước 5: Truy cập ứng dụng

Mở trình duyệt và truy cập: http://localhost:3000

V. HƯỚNG DẪN SỬ DỤNG

5.1. Trang đăng ký khám (index.html)

Tra cứu nhân viên:

  1. Nhập Mã nhân viên vào ô tìm kiếm
  2. Click nút "Tra cứu"
  3. Nếu tìm thấy: Thông tin nhân viên hiển thị và tự động điền vào form
  4. Nếu không tìm thấy: Hiển thị form đăng ký nhân viên mới

Đăng ký nhân viên mới (khi tra cứu không thấy):

  1. Điền đầy đủ thông tin (Họ tên bắt buộc)
  2. Click "Đăng ký nhân viên mới"
  3. Hệ thống sẽ thêm nhân viên và tự động chuyển sang đăng ký khám

Các tab khám:

  • Tổng quát: Chiều cao, cân nặng, BMI, mạch, huyết áp, phân loại thể lực
  • Nội khoa: Kết quả khám nội khoa
  • Ngoại/Da: Kết quả khám ngoại khoa và da liễu
  • Sản phụ: Kết quả khám sản phụ khoa (chỉ hiển thị với nữ)
  • Mắt: Kết quả khám mắt
  • TMH: Kết quả khám tai mũi họng
  • RHM: Kết quả khám răng hàm mặt
  • Kết luận: Kết luận sức khỏe và chỉ định

5.2. Trang quản lý nhân viên (employee-manager.html)

  • Xem danh sách: Hiển thị bảng danh sách nhân viên với đầy đủ thông tin
  • Tìm kiếm: Nhập từ khóa vào ô tìm kiếm
  • Lọc: Chọn phòng ban từ dropdown
  • Thêm nhân viên: Click "Thêm NV" → Điền thông tin → Lưu
  • Sửa nhân viên: Click "Sửa" → Chỉnh sửa → Lưu
  • Xóa nhân viên: Click "Xóa" → Xác nhận
  • Import Excel: Click "Import Excel" → Chọn file Excel
  • Export Excel: Click "Xuất Excel" → Tải file
  • Chuyển sang khám: Click "Khám" → Chuyển sang trang đăng ký
